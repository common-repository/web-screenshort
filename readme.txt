== Description ==

This plugin allows any WordPress user to easily add thumbnail previews of websites right in the content of their posts using a simple shortcode [web-screenshort url="http://www.wordpress.org/" width="500" height="400" alt="WordPress Blog Tool, Publishing Platform, and CMS"]

**Video Demo**

http://www.youtube.com/watch?v=xcJjLTestgI

== Installation ==

= First download the ZIP file =
1. Log in to your website administrator panel.
2. Go to Plugins page, and add new plugin.
3. Upload [Web Screenshort](https://wordpress.org/plugins/web-screenshort/).
4. Click Install Now button.
5. Then click Activate Plugin button.

or

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Web Screenshort' and install it 
3. Head over to **Post >> Add new** or **Page >> Add new**
4. Click **WP DEV** button on tinymce editor 
5. now you will see a popup window just fill detail for your screenshot and press ok

**That's it** 
**Now visit your site and see screenshot of website**
